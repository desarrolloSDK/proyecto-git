Comandos Básicos de Git

  • git init - Inicializa un nuevo repositorio de Git.
  • git clone [url] - Clona un repositorio remoto a tu máquina local.
  • git status - Muestra el estado de los cambios en tu repositorio.
  • git add [archivo] - Añade un archivo al área de preparación.
  • git commit -m "mensaje" - Realiza un commit de los cambios en el área de preparación.
  • git push - Envía los commits a un repositorio remoto.
  • git pull - Actualiza tu repositorio local con los cambios del repositorio remoto.
  • git branch Lista de todas las ramas. !!
